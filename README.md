# TitleCat

咸鱼之作

  一款用来批量获取网站标题的程序，本来只是临时写出来用的，结果发现作用挺大的，就把自己的想法稍微写了点出来了。

功能：导出表格，使用socket5代理，指定线程，自定义UA头，自定义超时时间。
